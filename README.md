<h1 align = "center">Windows Local User Management Toolkit (PowerShell)</h1>
Descripción

Herramienta desarrollada en PowerShell para la administración de usuarios locales en Windows mediante un menú interactivo.

El proyecto permite identificar usuarios autorizados antes de acceder al sistema y realizar tareas básicas de administración utilizando los cmdlets nativos de PowerShell.

Características
Autenticación básica de acceso al script.
Menú interactivo en consola.
Creación de usuarios locales.
Comprobación de usuarios existentes.
Eliminación de usuarios locales.
Interfaz sencilla con mensajes en color.
Organización del código mediante funciones.
Tecnologías utilizadas
PowerShell
Windows
Cmdlets de administración local (New-LocalUser, Get-LocalUser, Remove-LocalUser)
Requisitos
Windows 10/11
PowerShell 5.1 o superior
Ejecutar el script como administrador
Uso
Ejecutar el script.
Introducir un usuario autorizado.
Seleccionar una opción del menú.
Administrar los usuarios locales del equipo.
Objetivo del proyecto

Practicar la automatización de tareas de administración en Windows utilizando PowerShell y los cmdlets nativos para la gestión de usuarios locales.
